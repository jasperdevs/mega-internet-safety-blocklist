# Source Notices

Mega Internet Safety Blocklist is generated as both a uBlock-style filter list and a plain DNS domain export. It combines recovered local filter data with maintained public domain blocklists that match the same safety-filtering scope.

The generated output is distributed under GPL-3.0 because it includes GPL-3.0 source lists.
