# ⚡ Mega uBlock Origin Safety Blocklist

![domains](https://img.shields.io/badge/domains-199%2C191-brightgreen)
![format](https://img.shields.io/badge/format-uBlock%20Origin-2ea44f)

A broad uBlock Origin filter list for stricter, safer browsing.

This repository contains filter rules for user-controlled content filtering only. It does not host, embed, mirror, or promote third-party media or services.

## Install

Raw list:

`https://raw.githubusercontent.com/jasperdevs/mega-ublock-origin-safety-blocklist/main/mega-ublock-origin-safety-blocklist.txt`

1. Open the uBlock Origin dashboard.
2. Go to **My filters**.
3. Enable **My custom filters**.
4. Open the raw list URL and copy the filter rules.
5. Paste the rules into the textbox.
6. Click **Apply changes**.

## Scope

- Focused on adult and graphic shock-content sites.
- Includes compact hostname regex rules for large adult/shock domain families.
- Excludes known non-scope categories such as gambling, drugs, alcohol, weapons, and generic security feeds.
- Deduplicated and sorted.
- Intentionally broad, so false positives are still possible.

If a site breaks, disable this list temporarily or add a local exception in uBlock Origin.
